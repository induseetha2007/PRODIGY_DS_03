# PRODIGY_DS_03

## Task 3: Decision Tree Classifier

### Objective
Build a Decision Tree Classifier to predict whether a customer will purchase a product or subscribe to a service based on demographic and behavioral data.

### Dataset
Bank Marketing Dataset

### Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

### Project Workflow
- Imported and explored the dataset
- Performed data preprocessing and handled categorical variables
- Split the dataset into training and testing sets
- Trained a Decision Tree Classifier
- Evaluated the model using accuracy score and confusion matrix
- Visualized the Decision Tree to understand the classification process

### Files Included
- Task3.ipynb
- bank.csv

### Outcome
Successfully developed a Decision Tree Classification model to predict customer responses based on the Bank Marketing dataset. The project demonstrates the complete machine learning workflow, including data preprocessing, model training, evaluation, and visualization.
